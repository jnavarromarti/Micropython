# Cloto - Gestor de Tareas con Prioridad Inteligente

**Cloto** es un gestor de tareas inspirado en la mitología griega, diseñado para organizar y priorizar tus actividades diarias de manera eficiente. 
Este proyecto permite guardar, visualizar, y priorizar tareas según su fecha límite y nivel de importancia, además de generar reportes diarios.

## Características

- **Añadir Tareas:** Permite guardar tareas con descripción, fecha límite y nivel de prioridad.
- **Priorizar Tareas:** Ordena automáticamente las tareas según su urgencia.
- **Guardar y Cargar Tareas:** Las tareas se guardan en un archivo JSON para retomar el trabajo en cualquier momento.
- **Generar Reporte Diario:** Crea un archivo con el resumen de las tareas completadas y pendientes.
- **Extensible:** Preparado para futuras mejoras como predicción del tiempo estimado para cada tarea.
